# 🌸 TODO-my-dashboard (中文版)

一个基于 HTML/JS 的唯美待办仪表盘，支持时间轴拖拽与长期目标拆解。

## 🚀 快速部署
1. **Fork** 本项目到你的 GitHub。
2. **修改数据**：打开 `schedule.js` 修改你的课程。
3. **开启网页**：Settings -> Pages -> Branch 选择 `master` -> Save。

## ⚙️ 个性化配置 (无需写代码)
1. 访问你的网址后，点击左下角的 **"制作我的课表"**。
2. 生成并下载 `config.json`。
3. 回到首页，点击 **"恢复"** 按钮上传该文件即可。
