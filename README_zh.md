# 🌸 TODO-my-dashboard (中文版)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[English Version](./README.md)

一个基于 **HTML/CSS/JS** 的轻量级个人效率仪表盘。它将**任务池、时间轴与长期计划**完美结合，特别适合大学生管理课程表、考试 DDL 以及日常习惯。

## ✨ 功能亮点

- **📅 每日任务池**：支持按星期切换。任务在未排程或未完成前会一直保留在池中。
- **⏱️ 交互式时间轴**：支持从任务池直接**拖拽任务**到时间轴。支持**非整数时间**（如 10:30），渲染精确到分钟。
- **🚀 长期计划**：记录大目标并设置 **DDL**。自带进度条，支持一键**拆解**为今日任务。
- **⚙️ 内置排课引擎**：无需修改代码！可视化添加课程或习惯，支持按**工作日、周末或自定义多选**进行批量排程。
- **🎁 积分商店**：完成任务赚取积分，在商店兑换自定义的小奖励，让自律更有动力。
- **🌐 全面双语**：中英文一键切换，包括所有占位符、弹窗和日期格式。
- **🌸 唯美视觉**：毛玻璃 UI 风格，淡紫色渐变背景，配有动态樱花飘落效果。

## 🚀 快速部署与使用

1. **Fork 本项目**到你的 GitHub 账号。
2. **开启网页服务**：
   - 进入仓库的 **Settings** -> **Pages**。
   - 分支选择 `master` (或 `main`)，点击 **Save**。
3. **零代码个性化**：
   - 访问生成的链接，点击左下角的 **“⚙️ 我的课表管理”**。
   - 直接在弹窗里填入你的课表和习惯。
   - 使用 **“备份数据”** 功能将你的配置导出为 JSON 文件。

## 📂 项目结构

- `index.html` — 页面结构与弹窗。
- `style.css` — 唯美 UI 样式、布局与动画。
- `script.js` — 核心逻辑：拖拽引擎、积分系统、多语言适配及排课功能。
- `schedule.js` — 首次运行时的默认示例数据。
