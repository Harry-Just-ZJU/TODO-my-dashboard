# 🌸 TODO-my-dashboard

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[中文说明 (Chinese Version)](./README_zh.md)

A lightweight **personal productivity dashboard** (Task Pool + Timeline + Long-term Plans) built with **HTML/CSS/JavaScript**. Especially designed for students to manage classes, assignments, and study habits with a beautiful aesthetic.

## ✨ Key Features

- **📅 Daily Task Pool**: Switch between days of the week. Tasks stay in the pool until scheduled or completed.
- **⏱️ Interactive Timeline**: Drag and drop tasks from the pool into specific time slots (08:00 - 23:00). Supports **minute-level precision**.
- **🚀 Long-term Plans**: Track major goals with **Deadlines (DDL)**. Includes a progress bar and one-click **decomposition** into daily tasks.
- **⚙️ Schedule Engine**: Built-in visual editor to manage your classes/routines. Supports **batch adding** for workdays, weekends, or custom days.
- **🎁 Score & Reward System**: Earn points by completing tasks and spend them in the "Reward Store" to motivate yourself.
- **🌐 Bilingual UI**: Instant toggle between English and Chinese, with all UI elements perfectly localized.
- **🌸 Aesthetic UI**: Glassmorphism design with a soft lavender gradient and animated cherry blossom petals.

## 🚀 Quick Start (Deployment)

1. **Fork this repo** to your GitHub account.
2. **Enable GitHub Pages**:
   - Go to **Settings** -> **Pages**.
   - Select the `master` (or `main`) branch and click **Save**.
3. **Personalize without coding**:
   - Open your live site, click the **"⚙️ Schedule Settings"** button.
   - Use the visual editor to add your classes/routines.
   - Use the **"Backup"** button to export your data for safekeeping.

## 📂 Project Structure

- `index.html` — Main UI structure and modals.
- `style.css` — Modern glassmorphism styles and layout.
- `script.js` — Core logic: Drag-and-drop, scoring, i18n, and schedule engine.
- `schedule.js` — Default template data for the first-time run.
